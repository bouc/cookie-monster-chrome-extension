# Privacy Policy for Cookie Monster Chrome Extension

**Last Updated:** November 19, 2024

## Overview
Cookie Monster is a developer tool designed for viewing and exporting website cookies for testing and debugging purposes.

## Data Collection
**We do NOT collect, store, or transmit any user data.**

### What the Extension Does:
- Reads cookies from websites only when you click the "Extract" button
- Temporarily stores cookie data locally in your browser using Chrome's local storage API
- Does NOT send any data to external servers
- Does NOT track your browsing activity
- Does NOT collect personal information

### Local Storage Only
All cookie data extracted by this extension:
- Remains on your device only
- Is stored temporarily in Chrome's local storage
- Can be cleared at any time by clicking the "Clear" button
- Is never transmitted over the internet
- Is never shared with third parties

## Permissions Explained

### Active Tab
- **Why we need it:** To access the current website's URL when you use the extension
- **What we do:** Only read the domain name to fetch cookies for that specific site
- **What we DON'T do:** We don't track which websites you visit

### Cookies
- **Why we need it:** To read cookie values from websites
- **What we do:** Extract cookies only when you click "Extract"
- **What we DON'T do:** We don't modify, delete, or share cookies

### Storage
- **Why we need it:** To temporarily save extracted cookie data in your browser
- **What we do:** Store data locally so it persists when you reopen the extension
- **What we DON'T do:** We don't sync data to the cloud or other devices

## Third-Party Access
This extension does NOT:
- Share data with third parties
- Use analytics or tracking services
- Display advertisements
- Connect to external servers
- Require user accounts or registration

## Data Security
Since all data remains local on your device:
- No data transmission means no interception risk
- No servers means no data breaches
- You have full control over your data

## User Control
You can:
- Clear all stored data by clicking the "Clear" button
- Uninstall the extension to remove all local data
- Use the extension only when needed

## Changes to This Policy
We may update this privacy policy from time to time. Any changes will be reflected in the extension's GitHub repository with an updated "Last Updated" date.
